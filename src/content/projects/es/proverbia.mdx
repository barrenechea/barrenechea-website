[Proverbia](https://proverbia.net) es una colección española de frases célebres, citas, proverbios y dichos organizados por tema y autor. Es un sitio web gratuito para que lo visites, pero no ofrecen ninguna API para su uso.

Quería desarrollar un bot de Telegram que entregara la cita del día de Proverbia al usuario una vez que este enviara un mensaje. Por lo tanto, necesitaba una forma limpia de consultar sus citas de manera programática.

Y así, este proyecto hace precisamente eso y es un ejemplo rápido y amigable para demostrar las capacidades de Puppeteer.
