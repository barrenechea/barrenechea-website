[Proverbia](https://proverbia.net) ist eine spanische Sammlung berühmter Zitate, Sprichwörter, Redewendungen und Aphorismen, die nach Thema und Autor organisiert ist. Es ist eine kostenlose Website, die Sie besuchen können, aber sie bieten keine API für deren Nutzung an.

Ich wollte einen Telegram-Bot entwickeln, der das Zitat des Tages von Proverbia an den Benutzer liefert, sobald sie eine Nachricht gesendet haben. Daher brauchte ich eine saubere Möglichkeit, ihre Zitate programmatisch abzufragen.

Und genau das macht dieses Projekt und ist ein schnelles und freundliches Beispiel, um die Fähigkeiten von Puppeteer zu demonstrieren.
