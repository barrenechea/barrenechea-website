[Proverbia](https://proverbia.net) ist eine spanische Sammlung von berühmten Zitaten, Sprüchen, Sprichwörtern und Redewendungen, die nach Thema und Autor organisiert sind. Es ist eine kostenlose Website für Ihren Besuch, aber sie bieten keine API für deren Nutzung an.

Ich wollte einen Telegram-Bot entwickeln, der dem Benutzer das Zitat des Tages von Proverbia liefert, sobald dieser eine Nachricht sendet. Daher benötigte ich eine saubere Möglichkeit, ihre Zitate programmatisch abzufragen.

Und genau das macht dieses Projekt und es ist ein schnelles und benutzerfreundliches Beispiel, um die Fähigkeiten von Puppeteer zu demonstrieren.
