[Proverbia](https://proverbia.net) é uma coleção espanhola de frases famosas, citações, provérbios e ditados organizados por tema e autor. É um site gratuito para você visitar, mas eles não oferecem nenhuma API para seu uso.

Eu queria desenvolver um bot do Telegram que entregasse a citação do dia do Proverbia ao usuário assim que ele enviasse uma mensagem. Portanto, eu precisava de uma maneira limpa de consultar suas citações de forma programática.

E assim, este projeto faz exatamente isso e é um exemplo rápido e amigável para demonstrar as capacidades do Puppeteer.
