Si has estado por aquí antes, este sitio web ha tenido el mensaje de "hola" por algunos años. ¡Pero han cambiado hartas cosas con el tiempo!

El sitio web allá por el 2016 estaba alojado en un droplet de [DigitalOcean](https://digitalocean.com). Más tarde, decidí abrir puertos dentro de mi propia red y usé [DockerHub](https://hub.docker.com/r/barrenechea/barrenechea-website) para correr el sitio web como un contenedor en mi servidor [Unraid](https://unraid.net/es).

Antes de cambiar a despliegues en Edge, usé un [Túnel de Cloudflare](https://www.cloudflare.com/es-es/products/tunnel/) para aliviar la naturaleza de IP dinámica de mi ISP y reforzar los "puntos de accesibilidad" de mi red local.

Lo que estás leyendo ahora es una reconstrucción del sitio web desarrollada con [Astro](https://astro.build). Su capacidad de desplegarse en Edge (actualmente estoy usando la [red perimetral](https://www.cloudflare.com/es-es/learning/cdn/glossary/edge-server/) de Cloudflare) es una enorme mejora tanto desde la perspectiva de seguridad como del rendimiento resultante para los usuarios finales (¡tú!).

La fuente que elegí para este sitio web es [Work Sans](https://fonts.google.com/specimen/Work+Sans).
